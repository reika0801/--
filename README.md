# 数当てゲーム

コンピュータが選んだ数を当てるシンプルな数当てゲームです。

## 機能

- 🎯 ランダムな数を生成（難易度選択可能）
- 📊 試行回数のカウント
- 📝 推測履歴の表示
- 💡 ヒント表示（大きい/小さい）
- 🎉 勝利時の評価メッセージ
- 🔄 再プレイ機能

## 使い方

### 実行方法

```bash
python3 number_guessing_game.py
```

または

```bash
python number_guessing_game.py
```

### ゲームの流れ

1. 難易度を選択します（簡単/普通/難しい）
2. コンピュータが選んだ数を推測します
3. ヒントに基づいて次の推測を行います
4. 正解するまで繰り返します
5. ゲーム終了後、再プレイするか選択できます

### 難易度

- **簡単**: 1-50の範囲
- **普通**: 1-100の範囲
- **難しい**: 1-200の範囲

## 要件

- Python 3.6以上
